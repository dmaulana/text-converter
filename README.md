### Installation

```sh
$ npm install -g
```

### How to use

```sh
Usage: convert [options]

Options:
  --base64-hex <string>    base64 to hex
  --base64-ascii <string>  base64 to ascii
  --base64-utf <string>    base64 to utf-8
  --hex-base64 <string>    hex to base64
  --hex-ascii <string>     hex to ascii
  --hex-utf <string>       hex to utf-8
  --ascii-base64 <string>  ascii to base64
  --ascii-hex <string>     ascii to hex
  --ascii-utf <string>     ascii to utf-8
  --utf-base64 <string>    utf-8 to base64
  --utf-hex <string>       utf-8 to hex
  --utf-ascii <string>     utf-8 to ascii
  -h, --help               read more information
```
